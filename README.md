# Zip Cracker

## Installation

## Clone the Repository
```bash
git clone https://github.com/whitehatboy005/Zip-Cracker
```
## Move the file
```bash
cd Zip-Cracker
```
## Install Dependencies
```bash
pip install -r requirements.txt
```
## Run the Program
```bash
python crack.py
```
